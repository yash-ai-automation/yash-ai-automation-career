**Company:** [[virtusa-full-stack-developer]]

## Overview

### Payment Gateways

- Achieved **40% faster settlement** through optimized API flows
- Batch settlement processing: **85,000 records nightly**

### Data and Compliance
- SQL query optimization yielding **52% faster** compliance reports
- ETL pipelines reducing **85% of manual data entry**
