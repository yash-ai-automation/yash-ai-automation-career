# Target Roles

Yash Anghan's target role archetypes, ordered by fit. Each archetype maps to a different buyer persona -- understanding what each employer is looking for drives resume tailoring ([[ats-optimization]]) and interview preparation.

## Primary Targets

### 1. AI Automation Engineer
**What the buyer wants**: Someone who can design and build automated workflows powered by AI -- not just connect apps, but architect reliable systems with proper error handling and measurable ROI.

**Yash's fit**: Core identity. [[ai-automation]] experience with [[n8n]], [[make-com]], custom agents. Freelance track record ($43K savings, 520+ hrs/yr). Enterprise foundation from [[morningstar]], [[bell-canada]], [[virtusa]].

### 2. AI Engineer
**What the buyer wants**: Build and deploy AI-powered features in production. LLM integration, RAG, embeddings, inference at scale.

**Yash's fit**: Morningstar work -- [[generative-ai]] pipelines, 25K+ daily predictions at sub-200ms, [[rag-pipelines]], [[deepeval]] monitoring. See [[morningstar-software-engineer]].

### 3. Generative AI Engineer
**What the buyer wants**: Specialist in LLMs, prompt engineering, RAG, fine-tuning. Build GenAI products.

**Yash's fit**: LLM summarization, 94% classification accuracy, RAG over 15K+ documents, hallucination detection. Deep [[generative-ai]] experience.

### 4. ML Engineer
**What the buyer wants**: Train, deploy, and monitor ML models. MLOps discipline. Production inference.

**Yash's fit**: [[mlops-llmops]] -- DeepEval frameworks, AWS inference infrastructure, drift detection, model performance monitoring. 25% reliability improvement.

## Secondary Targets

### 5. AI Software Engineer
**What the buyer wants**: Software engineer who can integrate AI into existing systems. Strong coding + AI literacy.

**Yash's fit**: 6+ years full-stack (Java, Python, React, Node.js) plus AI/ML overlay. [[enterprise-engineering]] + [[generative-ai]] combination.

### 6. Agentic AI Engineer
**What the buyer wants**: Build autonomous AI agents that plan, execute, and iterate. Emerging role.

**Yash's fit**: [[workflow-orchestration]] experience designing multi-step autonomous processes. Custom agent development. Pipeline automation expertise.

## Excluded Roles (see [[deal-breakers]])
- Prompt Engineer (too narrow, underpays)
- Director+ / Principal-level (over-scoped)
- Internships (under-scoped)
- Pure backend/frontend without AI component
