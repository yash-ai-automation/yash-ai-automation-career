# Azure AI Engineer Associate

**Issuer:** Microsoft
**Certification:** Microsoft Certified: Azure AI Engineer Associate
**Level:** Associate (intermediate)

## Overview

The Azure AI Engineer Associate certification validates the ability to design and implement AI solutions using Azure Cognitive Services, Azure OpenAI Service, and other Azure AI platform components. It covers:

- Azure AI services (vision, speech, language, decision)
- Azure OpenAI Service integration
- Knowledge mining and document intelligence
- Conversational AI and bot frameworks
- Responsible AI practices
