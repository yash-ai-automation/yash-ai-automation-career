# Azure AI Fundamentals

**Issuer:** Microsoft
**Certification:** Microsoft Certified: Azure AI Fundamentals
**Level:** Fundamentals (foundational)

## Overview

The Azure AI Fundamentals certification (AI-900) validates foundational knowledge of AI and machine learning concepts and their implementation on Azure. It covers:

- Core AI and ML concepts
- Computer vision workloads on Azure
- Natural Language Processing (NLP) workloads
- Generative AI concepts
- Azure AI services overview
