# Spring Boot

Spring Boot is a Java-based framework for building production-grade microservices and web applications. It provides auto-configuration, embedded servers, and opinionated defaults that simplify enterprise Java development.

## Yash's Experience

Spring Boot was Yash's primary backend framework across two enterprise roles: [[bell-canada]] and [[virtusa]]. It underpins his 6+ years of full-stack Java development experience.

### Usage at Bell Canada
- Developed microservices architecture for real-time billing calculations handling 45K daily transactions
- Implemented REST APIs serving 850K active subscribers with sub-100ms response times
- Architected event-driven order management system using message queues
- Built full-stack monitoring dashboards with Java backend
- Enterprise-scale telecommunications systems with 99.9% uptime requirements

### Usage at Virtusa
