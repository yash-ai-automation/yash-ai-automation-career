# Playwright

Playwright is a browser automation framework by Microsoft for end-to-end testing, web scraping, and browser-based automation. It supports Chromium, Firefox, and WebKit, and offers both UI and API testing capabilities.

## Yash's Experience

Yash used Playwright at [[morningstar]] for automated testing of GenAI systems, and it is also a core tool in the career-ops pipeline itself.

### Usage at Morningstar
- Orchestrated automated API testing suites using Playwright and GitHub Actions to validate GenAI outputs
- Reduced regression cycles by 60%
- Ensured high-quality delivery across complex enterprise software ecosystems
- Validated that AI-generated content met accuracy and format requirements

### Usage in career-ops
- Used for HTML-to-PDF generation of tailored CVs (via `generate-pdf.mjs`)
- Used for web scraping and offer verification (navigating to job URLs, taking snapshots)
- Core to the pipeline's automated workflow

### Key Capabilities Leveraged

- **Browser automation:** Navigating web pages, capturing content
